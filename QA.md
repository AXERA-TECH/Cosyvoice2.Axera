# 常见问题  
1. 模型导出报错  
（1） 首先检查transformers版本是否在 4.40.1 以上  

2. 模型加载失败  
（1）传入的模型路径是否正确  
（2）CMM内存是否足够  


3. 系统卡死或崩溃  
（1）检查OS内存是否过低   


4. 模型转换失败  
（1）检查工具链版本  
需要 pulsar2 4.2 或更高版本  

5. 工具链报错  
（1）检查docker版本  
最低版本 Docker version 20.10.7, build f0df350  
Docker版本向下兼容​，新版本的 Docker 引擎通常能够运行旧版本 Docker 构建的镜像。  
但是如果Docker引擎版本太低工具链可能会报错。
